Overview

Notifies the application of the current driver distraction state (whether driver distraction rules are in effect, or not).

HMI Status Requirements:

    HMILevel:
    • Can be sent with FULL, LIMITED or BACKGROUND
    AudioStreamingState:
    • Any
    SystemContext:
    • Any

@since SDL 1.0
